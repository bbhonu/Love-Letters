  • SET UP
  • 1. Select “new repository” from github.com and give it a name
  • 2. To create a repository from scratch: enter this in your chosen directory line by line
- echo “# nameofrepository”>> README.md
- git init
- git add README.md
- git commit -m “first commit”
- git remote add origin https://github.com/(profilename)/(
nameofrepository).git
- git push -u origin master

OR push an existing repository to the new one
- git remote add origin https://github.com/(profilename)/(nameofrepository).git
- git push -u origin master

  • 3. git push: after committing, this will publish the file(s) to github !
